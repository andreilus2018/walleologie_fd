# Flying Device
Code for flying device and it's remote control, which using Arduino Nano and nRF24L01.

Connections:

HM-10 BLE:
5V - 5V
GND - GND
2 - RX
3 - TX

Servo:
5V - 5V
GND - GND
S - 10

Motor:
5V - 5V
GND - GND
S - 9

BMP280:
3.3V - 3.3V
GND - GND
SDA - A4
SCL - A5

2nd motor:
5V - 6
GND - GND
